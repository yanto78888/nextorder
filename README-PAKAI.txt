NEXTORDER UPDATE - PHONE MODE + FOTO PRODUK UPLOAD + BATAL DEPOSIT + UBAH ADMIN

Cara pakai:
1. Extract ZIP ini di root repo Nextorder kamu.
2. Pilih overwrite / replace semua file yang sama.
3. Jalankan: npm install
   Catatan: update foto produk memakai package multer.
4. Restart server.

Fitur yang sudah masuk:
- Tampilan phone mode dengan menu navigasi di bawah.
- Tombol Produk / Top Up / Riwayat / Profil / Admin dibuat lebih rapi untuk layar HP.
- Foto produk di admin memakai upload file, bukan URL.
- Folder upload: public/uploads/products.
- Batal deposit dari QR aktif dan dari riwayat top up.
- Halaman Admin > Setting punya form Akun Admin untuk ubah username dan password.

Untuk ubah username/password admin:
Admin > Setting > Akun Admin > isi password saat ini > isi username/password baru > Update Akun Admin.
